Work History

Backend Developer

IBM Canada

Sep 2021 - Sep 2023

During my time as a full-time employee at IBM Canada, my primary duties revolved around maintenance and enhancement of an automated release system. This release system was in charge of providing clients with the latest database updates related to vulnerability severity data, shipment of custom and industry vulnerability data, and updates to protocol and device management software. The main technologies of the project consisted of: Python3, Jenkins, Gitlab CD/CI, Docker, IBM Cloud, Kubernetes, Perl5, Bash, Postgres, and XML.
Moreover, during my last months at this employer I also contributed to another automated system used to test protocol software through the use of Jenkins CI, remote ssh access to virtual machines, and Groovy. This was a legacy system that was overdue for a revamp. Although my contributions to the implementation extended to only debugging major components of the system, I was involved in the Roadmap planning for the implementation. This included extensive meetings with my superiors to analyze best technical approaches, agile workflow, and analysis of business impact through data analytics using Jira data. The project aimed to reduce the time it took quality engineers to test complex protocols. Jira acted as a data source to record current testing times, categorization of testing times based on protocol types and overall backlog of testing tickets open.

  • JavaScript
